Konfiguracja EmailJS (darmowe, 200 e-maili/mies.)
1. Wejdź na emailjs.com → załóż konto
2. Dodaj usługę e-mail (Gmail) → skopiuj Service ID
3. Utwórz szablon → skopiuj Template ID
4. W zakładce Account skopiuj Public Key
5. W szablonie użyj: {{entry_name}} {{entry_date}} {{days_left}}
